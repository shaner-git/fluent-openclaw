# Fluent for OpenClaw v0.1.9

`fluent-openclaw` is Fluent's canonical public beta package for OpenClaw.

- Package version: `0.1.9`
- Minimum Fluent MCP contract: `2026-07-09.fluent-core-v2.0`
- Current reference Fluent MCP contract: `2026-07-09.fluent-core-v2.0`
- Public profile: 26 tools, 3 resources, 14 explicit writes, and 3 optional render adapters
- Required hosted scopes: `meals:read`, `meals:write`, `style:read`, `style:write`, `offline_access`

## What Changed

- Publishes the corrected Fluent 2.0 package as a new immutable version after the existing ClawHub `v0.1.8` artifact retained the previous release's README and contract metadata.
- Hosted OAuth requests `style:write` alongside the current Meals and Style scopes.
- Public guidance reflects scoped self-serve early access instead of invite-only access.
- Meals and Style are the active public domains.
- Budgets remains limited to user-declared `meals-groceries` and `style-clothing` envelope/spend context.
- Health/Wellbeing remains reserved and is not advertised as an active public domain.
- Retired Home, recipe-card, pantry-dashboard, Style setup, purchase-analysis, and standalone Visual Sync surfaces are not described as current behavior.

## Install

```bash
openclaw plugins install clawhub:fluent-openclaw
```

## Managed Early-Access Setup

```bash
openclaw fluent auth login
openclaw fluent mcp --track cloud
openclaw fluent doctor
openclaw fluent deep-check
```

Managed account creation is scoped self-serve while launch controls allow it and is currently free. Plugin-managed hosted auth remains beta.

## Run Fluent Yourself

```bash
openclaw fluent mcp --track oss --base-url http://127.0.0.1:8788 --token <oss-token>
openclaw fluent doctor --track oss --base-url http://127.0.0.1:8788 --token <oss-token>
```

The open-source runtime does not depend on managed early-access eligibility.

## Package Boundary

This repository is the canonical source for `fluent-openclaw`. The helper under `fluent-oss/openclaw-plugin/fluent` is exported from `fluent-mcp/openclaw-plugin/fluent`; it is a distinct package line and is not generated from this package line.

Browser execution helpers are intentionally omitted. Retailer automation helpers are intentionally omitted.
